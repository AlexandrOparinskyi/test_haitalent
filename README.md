# Тестовое задание Хайталант

## О проекте

Это RESTful API, разработанный с использованием FastAPI, предназначенное для управления вопросами и ответами. Система позволяет пользователям создавать вопросы и прикреплять к ним неограниченное количество ответов.

Ключевые особенности:
•   Вопросы: Создание, получение списка всех вопросов, получение вопроса по ID, удаление вопросов.
•   Ответы: Создание ответов для конкретного вопроса, получение ответа по ID, удаление ответов.
•   Гибкость: К каждому вопросу может быть добавлено множество ответов.

### Технологический стек
•   **Фреймворк:** FastAPI
•   **База данных:** PostgreSQL
•   **ORM:** SQLAlchemy (с использованием Alembic для миграций)
•   **Асинхронный драйвер PostgreSQL:** asyncpg
•   **Контейнеризация:** Docker, Docker Compose

## Начало работы

Эти инструкции помогут вам развернуть и запустить проект на вашей локальной машине для разработки и тестирования.

### Предварительные требования

Для запуска проекта вам понадобится установленный Docker и Docker Compose.

•   [Docker Desktop](https://www.docker.com/products/docker-desktop) (включает Docker Compose)

### Установка и запуск

Проект разработан с учетом легкости развертывания с помощью Docker Compose.

1.  Клонируйте репозиторий
```
  git clone https://github.com/AlexandrOparinskyi/test_haitalent.git
  cd test_haitalent
```

2. Настройте переменные окружения:

  Создайте файл .env в корневой директории проекта, скопировав .env.example:
  
```
  cp .env.example .env
```

  Откройте .env и при необходимости измените значения. В основном это параметры подключения к базе данных.

  Пример .env:
  
```
  POSTGRES_USER=user
  POSTGRES_PASSWORD=password
  POSTGRES_DB=questions_db
  DB_HOST=db
  DB_PORT=5432
```

3. Запустите Docker Compose:

  
```
  docker-compose up --build -d
```
